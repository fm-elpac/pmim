## 环境变量

- `XDG_RUNTIME_DIR`

  运行目录.

  - 口令文件路径: `{XDG_RUNTIME_DIR}/pmim/server_token`

  - HTTP 监听端口号: `{XDG_RUNTIME_DIR}/pmim/port`

  - unix socket 文件路径 (默认): `{XDG_RUNTIME_DIR}/pmim/us`

- `PMIMS_US`

  指定 unix socket 文件路径.

- `PMIMS_PORT`

  指定 HTTP 监听端口.

- `PMIMS_ANDROID`

  Android 运行模式.

  ```
  env PMIMS_ANDROID=1 ...
  ```

- `PMIMS_DB`

  指定数据库目录.

- `HOME`

  用户的家目录.

  - 内置数据库 (默认): `~/.config/pmim/pmim_sys.db`

  - 用户数据库 (默认): `~/.config/pmim/pmim_user.db`

- `PMIMS_DEBUG`

  启用调试模式.
